# MX Herbal Beauty Powder Landing Page

A premium, single-page landing site for **MX — Herbal Beauty Powder**. Designed with a luxurious, modern aesthetic featuring fluid typography, smooth scrolling interactions, glassmorphism UI accents, and interactive WhatsApp integration.

## Features
- **Premium Typography**: Fine-tuned layout built on *Cormorant Garamond* and *DM Sans*.
- **Scroll-Reveal Animations**: Hardware-accelerated CSS animations driven by `IntersectionObserver`.
- **Glassmorphic Navigation**: A frosted-glass header with shrinking transitions on scroll.
- **Custom Vector Ornaments**: Handcrafted inline SVG icons and background spa watermarks.
- **Interactive Enquiry Form**: Interactive form with built-in validation that formats details and redirects straight to WhatsApp.
- **Floating WhatsApp Concierge**: Persistent chat widget with conversion-focused pulse animation.
- **SEO & Social Optimized**: OpenGraph meta descriptors, viewport constraints, and semantic hierarchies.
